# VIREL_duality
Dos perspectivas. Una sola esencia.🜂 VisiónVIREL_duality es un sistema digital diseñado para explorar la dualidad interna:
la luz y la sombra, la calma y la confrontación, la contemplación y la transformación.No busca cambiar quién eres.
Busca que conozcas cada parte de ti para vivir en equilibrio.
🜁 Los dos caminos✦ Camino ZenQuietud.
Respiración.
Luz.
Naturaleza.
Minimalismo.
Restauración.✦ Camino DualityConfrontación.
Preguntas difíciles.
Sombras.
Decisiones.
Crecimiento.
Transformación.Ambos caminos conducen al mismo destino: equilibrio.
 Lenguaje ritual: Sakura · Ki · IshiVIREL_duality utiliza un lenguaje conceptual propio:Sakura — fragilidad, belleza, impermanencia.Ki — energía vital, intención, movimiento interno.Ishi — piedra, estructura, fundamento.Estos conceptos funcionan como símbolos y herramientas de introspección.
🜄 El avatar como espejoEl sistema incluye avatares que representan distintas voces internas:Zen — guía silenciosa.Duality — confrontación honesta.Mirror — integración de ambas.El avatar no existe para decir “todo está bien”.
Existe para preguntar:“¿Actúas por miedo o por convicción?”“¿Esta decisión refleja quién quieres ser?”“¿Qué parte de ti está hablando ahora?”🜃 Tecnologías
 TecnologíasVIREL_duality está construido sobre:AstroTypeScriptArquitectura modularSistema de diseño ritual (tokens, themes, UI minimalista)Archivos conceptuales y avatares en JSON/MD
